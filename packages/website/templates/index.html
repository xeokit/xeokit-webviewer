@xeokit/sdk

Welcome to the next generation of the xeokit SDK!

This version of the SDK is a complete rewrite of the xeokit SDK in TypeScript. In this version, we've introduced several new key capabilities and redesigned the API following lessons learned from previous versions of the SDK. The result is a more modular, flexible and extensible toolkit that supports even more BIM programming use cases.

To learn more about this new SDK version, we recommend starting with our Technical White Paper.
