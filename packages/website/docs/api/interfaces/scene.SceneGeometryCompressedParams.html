Interface SceneGeometryCompressedParams

Pre-compressed geometry creation parameters for SceneModel.createGeometryCompressed.

  • Created from SceneGeometryParams using compressGeometryParams
  • Used with SceneModel.createGeometryCompressed
  • Generates edge indices for triangle meshes
  • Ignores normals (our shaders auto-generate them)
  • Quantizes positions and UVs as 16-bit unsigned integers

See @xeokit/sdk/scene for usage.
