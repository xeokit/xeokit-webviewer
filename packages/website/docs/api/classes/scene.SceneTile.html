Class SceneTile

A tile in a SceneModel.

  • SceneMeshParams.origin
  • Stored in SceneModel.tiles
  • Created automatically for each distinct value of SceneMeshParams.origin given to SceneModel.createMesh
  • Each SceneTile is destroyed as soon all SceneTiles with that origin heve been destroyed

See @xeokit/sdk/scene for usage.
