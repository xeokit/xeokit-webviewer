attachSceneModel

  • attachSceneModel(sceneModel: SceneModel): void | SDKError
  • Internal

    Attaches a SceneModel to this WebGLRenderer.

    This method attaches various hooks to the elements within the SceneModel, through which they can upload state updates to the Renderer.

    • Sets a scene!RendererModel on SceneModel.rendererModel
    • Sets a scene!RendererObject on each SceneObject.rendererObject
    • Sets a scene!RendererMesh on each SceneMesh.rendererMesh
    • Sets a scene!RendererTextureSet on each SceneTextureSet.rendererTextureSet
    • Sets a scene!RendererTexture on each SceneTexture.rendererTexture

    Then, when we make any state updates to those components, they will upload the updates into the Renderer.

    You must first attach a View with Renderer.attachView before you can attach a SceneModel.

    Parameters

    • sceneModel: SceneModel

    Returns void | SDKError

    void

    • SceneModel successfully attached.
