Class OrthoProjection

Orthographic projection configuration for a Camera .

  • Located at Camera.orthoProjection.
  • Works like Blender's orthographic projection, where the positions of the left, right, top and bottom planes are implicitly indicated with a single OrthoProjection.scale property, which causes the frustum to be symmetrical on X and Y axis, large enough to contain the number of units given by OrthoProjection.scale.
  • OrthoProjection.near and OrthoProjection.far indicated the distances to the clipping planes.
  • OrthoProjection.onProjMatrix will fire an event whenever OrthoProjection.projMatrix updates, which indicates that one or more other properties have updated.
