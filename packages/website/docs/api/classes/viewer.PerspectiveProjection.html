Class PerspectiveProjection

PerspectiveProjection projection configuration for a Camera .

Summary

  • Located at Camera.perspectiveProjection.
  • Implicitly sets the left, right, top, bottom frustum planes using PerspectiveProjection.fov.
  • PerspectiveProjection.near and viewer!PerspectiveProjection.far| PerspectiveProjection.far specify the distances to the clipping planes.
  • PerspectiveProjection.onProjMatrix will fire an event whenever PerspectiveProjection.projMatrix updates, which indicates that one or more other properties have updated.
