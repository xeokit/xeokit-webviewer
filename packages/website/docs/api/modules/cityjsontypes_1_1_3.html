Namespace cityjsontypes_1_1_3

xeokit CityJSON 1.1.3 Data Types

  • Defines numeric constants for the set of CityJSON 1.1.3 entity and relationship types.
  • Use with data to assign CityJSON types to DataObjects and Relationships and treat them as elements of a basic entity-relationship graph.
  • Use with treeview , to configure the appearance and behaviour of TreeViews for navigating CityJSON objects.
